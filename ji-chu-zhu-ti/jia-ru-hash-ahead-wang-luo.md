# 加入Hash Ahead网络

使用Hash Ahead网络，首先需要进行节点的配置，可参考[运行节点](jie-dian-yu-ke-hu-duan/yun-hang-jie-dian.md)

## 加入测试网

首先进入hashahead文件夹中

```
$ cd hashahead
```

然后运行测试网节点

```
$ hashahead -daemon -testnet
```

停止测试网的节点

```
hashahead>stop
```

返回：

```
hashahead server stopping
```

## 加入主网

首先进入hashahead文件夹中

```
$ cd hashahead
```

然后运行主网节点

```
$ hashahead -daemon
```

停止主网的节点

```
hashahead>stop
```

返回：

```
hashahead server stopping
```
